切片，就是列表的一部分，当明确知道从第x个元素切到第y个元素，示例为

```python

print(list_v[:4])# 从数组开头开始，切到索引4之前

```

当明确隔几个元素采样一次时，示例如下

```python
list_v = [1, 2, 3, 4, 5]

print(list_v[1:-1:2])# 切除一头一尾后，每2个元素采样一次
```

需要注意的是，经过切片后的列表是一个新的对象，和原本的对象独立，所以对存储造成了影响，为此，NumPy的切片被设定为是原对象的一个视图，不会在内存中创建一个新对象，改变Numpy切片上的对象会影响原Numpy数组

### 类

创建和使用类

+ 类的本质：在一堆函数之间传递参数
+ 根据约定，类的名称需要首字母大写
+ **类中的函数叫方法**，一个类包含一个\_\_init\_\_方法+很多自定义方法，**每个类中都必须包含\_\_init\_\_方法**

```python
        return self.a + self.b
    
    def sub(self):
        return self.a - self.b
```

继承

在某个类（父类）的基础上去添加几个方法，形成另一个类（子类）

+ 父类从无到有去写属性和方法，第一行是class类型
+ 子类可继承父类的属性和方法，第一行是class类名（父类名）

子类在特殊方法里使用`super()`函数，就可以继承父类的全部属性与方法

```python
# 子类 （在加减的基础上，添加乘除功能）
class Counter2(Counter):
    '''一台可以加减乘除的高计算器'''
    
    def __init__(self, a, b):
        '''引用父类的属性'''
        super().__init__(a, b) # 继承父类(superclass)
        
    def mul(self):
        return self.a * self.b
    
    def div(self):
        return self.a / self.b
```

如果想要在子类中修改父类中的某个方法，可以直接在子类里写一个同名方法，即可实现覆写，你也可以把覆写称为“变异”

掠夺

继承只能继承一个类的方法，但如果想要得到很多其它类的方法，则需要掠夺功能。有了掠夺功能，一个类可以掠夺很多其它的类，示例如下：

```python
# 掠夺者
class Amrc:
    '''一台本身只能乘除的计算器，现欲掠夺加减功能'''
    def __init__(self, c, d):
        self.c = c
        self.d = d
        self.cnt = Counter(c, d) # 掠夺Counter类的方法
```

## Numpy数组库

导入Numpy时，通常给其一个别名“np”，即`import numpy as np`

### 数据类型

列表需要额外空间存储数据类型，造成了存储空间的冗余，Numpy数组只能存储一种数据类型，就克服了这种缺点，以节省内存。为方便起见，可将Numpy数组简单分为整数型数组和浮点型数组

```python
import numpy as np

# 创建整数型数组
arr1=np.array([1, 2, 3]) # 元素若都是整数，则为整数型数组
print(arr1)

# 创建浮点型数组
arr2=np.array([1.1, 2, 3]) # 内含浮点数，则为浮点型数组
print(arr2)

[1 2 3]
[1.1 2.  3. ]
```

注意，使用print输出Numpy数组后，元素之间没有逗号，一是可以与Python列表区分开来，二是避免逗号与小数点之间的混淆

**同化定理**

+ 往整数型数组里插入浮点数，该浮点数会自动被截断为整数
+ 往浮点型数组里插入整数，该整数会自动升级为浮点数

**共同改变定理**

需要把所有的元素都改变，才能改变数组类型

整数型数组和浮点型数组

`arr2 = arr1.astype(float)`

除了上述方法，只要满足共同改变定理，整数型数组和浮点型数组仍然可以相互转换。最常见的是整数型数组在运算过程中升级为浮点型数组，示例如下。

```python
arr = np.array([1, 2, 3])

# 整数型数组与浮点数做运算
print(arr + 0.0)
print(arr * 1.0)

# 整数数组遇到触发（即便是除以整数）
print(arr / 1)

# 整数型数组与浮点型数组做运算
int_arr = np.array([1, 2, 3])
float_arr = np.array([1.0, 2, 3])
print(int_arr + float_arr)
```

### 数组维度

主要学习一维和二维数组

+ 不同维度的数组之间，从外形上的本质区别是
	+ 一维数组使用1层中括号表示
	+ 二维数组使用2层中括号表示
	+ 三维数组使用3层中括号表示
+ 有些函数需要传入数组的形状参数，不同维度数组的形状参数为：
	+ 一维数组的形状参数形如：`x` 或 `(x,)`；
	+ 二维数组的形状参数形如：`(x,y)`；
	+ 三维数组的形状参数形如：`(x,y,z)`。
+ 现在以同一个序列进行举例：
	+ 当数组有 1 层中括号，如 `[1 2 3]`，则其为**一维数组**，其形状是 `3` 或 `(3,)`；
	+ 当数组有 2 层中括号，如 `[[1 2 3]]`，则其为**二维数组**，其形状是 `(1,3)`；
	+ 当数组有 3 层中括号，如 `[[[1 2 3]]]`，则其为**三维数组**，其形状是 `(1,1,3)`。

这里用后面要讲的 `np.ones()` 函数进行演示，只因其刚好需要传入形状参数。

```python
arr1 = np.ones(3)
arr2 = np.ones((1, 3))
arr3 = np.ones((1, 1, 3))

print(arr1.shape)
print(arr2.shape)
print(arr3.shape)
```

**不同维度数组之间的转换**

一维数组转二维数组，还是二维数组转一维数组，均要使用的是数组的重塑方法 `.reshape()`，该方法需要传入重塑后的形状（shape）参数。

这个方法神奇的是，给定了其他维度的数值，剩下一个维度可以填 `-1`，让它自己去计算。比如把一个 5 行 6 列的矩阵重塑为 3 行 10 列的矩阵，当列的参数 `10` 告诉它，行的参数直接可以用 `-1` 来替代，它会自己去用 `30` 除以 `10` 来计算。

```python
arr1 = np.arange(10)

# 升级为二维数组
arr2 = arr1.reshape((1, -1))

# 降级为一维数组
arr3 = arr2.reshape(-1)
```

### 数组的索引

**访问数组元素**

和正常数组一样，用`[]`

**花式索引**

向量的花式索引

```python
arr1 = np.arange(0, 100, 10)
print(arr1[[0, 2]]) # [ 0 20]
```

| 索引方式 | 向量                   | 矩阵                                   |
| -------- | ---------------------- | -------------------------------------- |
| 普通索引 | `arr1[x₁]`             | `arr2[x₁, y₁]`                         |
| 花式索引 | `arr1[[x₁,x₂,...,xₙ]]` | `arr2[[x₁,x₂,...,xₙ], [y₁,y₂,...,yₙ]]` |

**访问数组切片**

向量`arr1[1:4]`

矩阵

**提取矩阵的行**


### 数组的变形

**数组的转置**

`.T`只对矩阵有效

`arr3 = arr2.T`

**数组的翻转**

数组的翻转方法有两个，一个是上下翻转的 `np.flipud()`，表示 up-down；一个是左右翻转的 `np.fliplr()`，表示 left-right。其中，向量只能使用 `np.flipud()`，在数学中，向量并不是横着排的，而是**竖着**排的。

**数组的重塑**

想要重塑数组的形状，需要用到 `.reshape()` 方法。

前面说过，给定了其他维度的数值，剩下一个维度可以填 `-1`，让它自己去计算。比如把一个 5 行 6 列的矩阵重塑为 3 行 10 列的矩阵，当列的参数 `10` 告诉它，行的参数直接可以用 `-1` 来替代，它会自己去用 `30` 除以 `10` 来计算。
